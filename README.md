# Quantium Retail Analytics Project

This project analyzes chip purchasing behaviour and evaluates the impact of store layout trials using transaction data.

## Project Overview

The analysis was conducted as part of the Quantium Data Analytics Job Simulation on Forage.

The objective was to:

- Understand customer purchasing behaviour
- Identify key customer segments
- Evaluate store trial performance
- Provide strategic recommendations

## Tasks Completed

### Task 1 – Customer Analytics

Analyzed transaction data to identify key customer segments and purchasing behaviour.

Key metrics examined:

- Total sales
- Units per customer
- Price per unit

Insights showed that family segments and mainstream customers drive the majority of chip sales.

---

### Task 2 – Experimentation and Uplift Testing

Evaluated store layout trials conducted in stores:

77, 86, and 88.

Control stores were selected based on similarity in historical sales trends.

Results:

- Store 77 → Significant uplift
- Store 86 → No significant change
- Store 88 → Significant uplift

---

### Task 3 – Business Recommendations

Prepared a client-style presentation summarizing insights and recommendations.

Key recommendation:

Roll out the layout strategy to additional stores while continuing to monitor performance.
